MEROGHAR QUICK START

1. Unzip this project.
2. Open only the MeroGharComplete folder in VS Code.
3. Open Terminal in VS Code.
4. Run: npm install
5. Run: npx expo start -c
6. Scan the QR code using Expo Go.
